Risks

Data Collection

Acquiring licenses for the data from Google, VicRoads, parking data from local councils, getting the machine learning to analyse probabilities correctly. Communicating other apps such as Uber may incur a clash in corporate policy and some network security concerns when sourcing to a third-party entity.

Software Developing

As we are all beginners of programming, it is very difficult for us to build an actual app that will work. We need to learn some program languages that we’ve never used before, which is a big risk for all of us. This is our first time trying to build an app.

icense problem

Most of our work is based on the tutorials provided on Google Maps Platform. Although there are a lot of open-source codes on the website for us to use, we can’t apply the APIs to make our app’s function more completed without significant time investment of learning how to code our own iOS or website map.
