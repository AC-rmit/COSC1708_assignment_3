Team members’ experience

For this project it is required to use software which can develop an application such as Figma or proto.io with Figma being chosen for the as the software that was going to be used to make a prototype of the application. No software licenses were needed as this is free and all assets used in the making of the design were made using free available images, logo, etc. Previously members had experience using Figma for other classes making prototypes of applications there. No group member had no prior experience with proto.io before starting this assignment. A few members do have experience in developing databases in SQLite setting, which will ease the flow of data for the application when it requires address and user information. Creation of SQLite databases is one of the skillsets for Aaron due to his work in Data Operations. In this project, most APIs are based on JavaScript, so we have to learn to program on JavaScript.
